期权对价格之和工作台。用户可以提供期权对来添加到工作台，或者直接打开工作台查看。

工作台脚本: `/Users/zhangxiaoyu/Scripts/price_sum_workbench.py`
配置文件: `/Users/zhangxiaoyu/Scripts/price_sum_pairs.json`
端口: 8052

## 参数解析

用户输入的参数 `$ARGUMENTS` 是期权对（可选），格式：
- `C37600+P17000`（简写，默认 ag2604）
- `C37600 P17000`（空格分隔）
- `ag2604C37600+ag2604P17000`（完整合约代码）
- 无参数：仅打开工作台

解析规则：
1. 如果用户只写了 `C37600+P17000`，默认补全为 `ag2604` 前缀
2. 如果指定了其他品种前缀（如 `cu2604C80000`），使用用户指定的

## 执行步骤

全程自动化，不要询问确认：

### 1. 检查工作台是否已在运行

```bash
pgrep -f price_sum_workbench.py
```

如果没有运行，启动它：
```bash
nohup /usr/bin/python3 /Users/zhangxiaoyu/Scripts/price_sum_workbench.py > /tmp/price_sum_workbench.log 2>&1 &
```
等待 3 秒确认启动成功。

### 2. 如果用户提供了期权对参数

把新期权对添加到配置文件 `price_sum_pairs.json`（读取现有配置，追加新对，避免重复）。
工作台会在下次刷新时自动加载新对。

### 3. 打开浏览器

```bash
open http://127.0.0.1:8052
```

### 4. 输出确认

告诉用户：
- 工作台地址 http://localhost:8052
- 如果添加了新对，告知已添加
- 提示可以在页面上直接添加/删除期权对
