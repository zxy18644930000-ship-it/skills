# 价格之和运动规律研究

对期权对价格之和进行数据分析，更新知识库中的信念，输出交易指导。

## 知识库

路径: `/Users/zhangxiaoyu/Scripts/price_sum_knowledge.json`

这是一个贝叶斯知识库，存储了我们对"期权对价格之和运动规律"的信念和参数估计。每次调用本技能都要读取它、用新数据验证、更新后写回。

## 执行流程

### 第1步：读取知识库

读取 `price_sum_knowledge.json`，了解当前所有信念及其置信度。

### 第2步：拉取最新数据

从本地数据库拉取数据进行分析。根据用户的需求（或默认行为）决定分析范围：

**默认行为**（无参数时）：分析当前/最近交易日的数据
**用户指定**：可以指定某个配对、某个时间段、或某个特定假设来验证

### 第3步：验证信念 & 贝叶斯更新

拿到第2步的数据后，**逐条对比知识库中的信念**：

对每个信念：
1. 判断今天的数据是**支持**、**反驳**、还是**无关**
2. 更新置信度：
   - 支持: `confidence = min(confidence + 0.05 * (1 - confidence), 0.99)` — 越接近1增长越慢
   - 反驳: `confidence = max(confidence - 0.15 * confidence, 0.01)` — 反驳权重是支持的3倍
   - 无关: 不变
3. 更新 evidence_for / evidence_against 计数
4. 追加 data_points

对参数估计（如净Delta）：
- 用增量平均更新: `new_value = old_value + (observed - old_value) / (samples + 1)`
- samples += 1

**判断标准示例**：
- B001(方向性): 期货涨且价格和涨→支持; 期货涨但价格和跌→反驳; 期货不动→无关
- B002(不对称性): 计算今日下跌敏感度/上涨敏感度比值，>1.5→支持，<1.0→反驳
- B003(虚值杠杆): 远端百分比波动>近端百分比波动→支持
- B006(弹簧效应): 价格和从低位反弹→支持; 低位继续走低→反驳

### 第4步：发现新信念

如果数据中出现**知识库里没有的规律**，添加为新信念，初始confidence=0.50。

例如可能发现：
- 夜盘vs日盘的价格和行为不同
- 某个时间段Theta衰减特别快
- 价格和与期货的关系在某个阈值处发生结构变化
- 波动率突变时价格和的非线性跳升

## 重要原则

1. **诚实标注不确定性**: confidence低就是低，不要因为理论上"应该"就人为抬高
2. **数据为王**: 任何信念都可以被数据推翻，不管它多"直觉正确"
3. **区分品种**: ag的规律不一定适用于其他品种，如果以后扩展到其他品种，需要单独的信念体系
4. **参数会漂移**: Delta、敏感度等参数随时间和期货价格变化，关注趋势而非绝对值
5. **少即是多**: 宁可维护5个高质量信念，不要堆50个模糊信念
