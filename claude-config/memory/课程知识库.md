### 学习笔记

#### 课时1: Gamma深度理解 (2026-03-03)
- Gamma=∂Delta/∂S，是Delta的加速度，永远为正
- ATM Gamma最大，DTE越短ATM附近Gamma越尖锐（针状）
- 卖出宽跨=Short Gamma（大波动加速亏损），买入宽跨=Long Gamma（大波动加速盈利）
- B008"gamma加速"现象: P17000从深OTM变为ATM时Gamma增大6倍，Delta从-0.05变为-0.48
- B018甜蜜点解释: DTE≤7远OTM = 高Theta + 低Gamma，远离Gamma峰值
- B013失败解释: 近ATM + 低DTE = Gamma爆发区，Theta无法覆盖Gamma亏损
- **系统缺口**: trade2026中0行Gamma代码，需加入GEX监控和Theta/Gamma比率
- **待办**: 在risk模块加入Gamma Exposure监控

#### 课时2: Gamma Scalping (2026-03-04)
- Gamma Scalping = Long Gamma + Delta对冲，赚的是**路径**而非位移
- 每次对冲="高卖低买"期货，利润 = ½×Gamma×ΔS²（平方项！不分方向）
- 成本是Theta，盈亏平衡波动 = √(2Theta/Gamma) = `/gamma`扫描的指标
- 白银实盘120分钟演示：Scalping +3351元 vs 纯持有 +1224元（2.7倍）
- 适合ATM+低DTE+震荡市；深OTM策略(现有strangle_buy)适合纯持有不适合对冲
- **关键洞察**: T/G比低=卖方噩梦、Scalper天堂。P棕榈油(T/G=1.3)极适合Scalp
- **与B006关联**: "弹簧效应"是被动版Gamma Scalping，主动对冲可更稳定
- **待办**: 考虑在trade2026中增加ATM Gamma Scalping策略类

#### 课时3: Vanna & Charm (2026-03-04)
- **Charm = ∂Delta/∂t**: 即使期货不动，Delta每天都在变。OTM期权的Delta随时间"蒸发"趋向0
- **Vanna = ∂Delta/∂σ**: IV变化直接改变Delta。IV升→OTM期权的Delta增大→期权"活过来"
- **Charm是卖方的朋友**: 时间流逝→Delta趋零→方向风险降低。B021策略(入DTE25→出DTE7)完美利用了Charm
- **Vanna是卖方的敌人**: IV飙升→Delta剧变→方向敞口突然增大→PnL放大(MAE来源)
- **B004翻转分解(白银C25000+P22000)**: Charm贡献-5.5%, Vanna贡献+6.2%, Gamma贡献-100.7%。期货暴跌(-11%)是压倒性主因，Charm和Vanna几乎互相抵消
- **CF vs SA的MAE差异根因**: CF IV波动大→Vanna效应大→Delta剧变→MAE 128%; SA IV稳定→MAE 27%
- **系统更新**: greeks_engine.py已添加vanna/charm/volga三个二阶Greeks字段
- **待办**: 在实时监控面板中显示Vanna/Charm，辅助卖出策略的持仓管理

#### 课时5: 波动率微笑 (Volatility Smile) (2026-03-09)
- **波动率微笑 = BS模型缺陷的市场补丁**: 同一到期日不同行权价IV不同，OTM期权IV更高
- **四个原因**: 肥尾(fat tails) + 波动率非恒定 + 跳跃风险(jumps) + Volga溢价(课时4关联)
- **三种形状**: U型微笑(商品)、负偏斜(股指)、正偏斜(某些农产品如CF)
- **DTE越短微笑越陡**: Volga/Vega比率随DTE下降而爆炸 → 做市商OTM溢价更高
- **CF vs SA微笑差异 = MAE差异根因**: CF微笑陡(翅膀+8-12bp) → MAE128%; SA平(+3-5bp) → MAE27%
- **卖出宽跨 = 卖微笑翅膀**: 收取尾部风险溢价，微笑越陡权利金越多但MAE也越大
- **高低腿比≤2 = 控制偏斜暴露**: 腿比过大说明微笑严重不对称，一侧风险不成比例
- **测量指标**: 25Δ Risk Reversal(偏斜方向) + 25Δ Butterfly(陡峭程度，MAE先行指标)
- **SVI模型**: w(k) = a + b[ρ(k-m) + √((k-m)²+σ²)], 5参数拟合整条微笑，课时7详学
- **系统对接**: iv_surface.py已有smile表格，建议升级为可视化+RR/BF指标+微笑监控
- **学习页面**: ~/Scripts/lesson5_smile.py → http://localhost:8060

#### 课时4: Volga (Vomma) (2026-03-06)
- **Volga = ∂Vega/∂σ = ∂²Price/∂σ²**: Vega对波动率自身的敏感度，即"波动率维度的加速度"
- **ATM Volga≈0, OTM Volga大**: ATM的Vega稳定(Volga/Vega≈0); 深OTM的Vega极不稳定(Volga/Vega>40)
- **DTE越低, Volga/Vega比率爆炸**: DTE=30时16.9, DTE=7时72.4, DTE=5时101.3
- **B020双涨的数学解释**: 同样IV+3%, DTE=15的Sum涨105%, DTE=60只涨42%. 原因: 低DTE的Sum基数小 + Volga/Vega高 → 正反馈加速
- **卖方Short Volga的非线性亏损**: IV+10%时实际亏损(-200)是线性外推(-90)的2.2倍, 差额110全是Volga造成
- **CF MAE 128%的完整解释**: Vanna(IV→Delta剧变) + Volga(IV→Vega膨胀) 双重放大
- **Volga解释波动率微笑**: 做市商为OTM期权的Volga(凸性)风险收取溢价 → OTM报价IV更高
- **第一阶段完成**: Delta→Gamma→Vanna→Charm→Volga, 完整的一阶+二阶Greeks认知框架
- **待办**: 在stop_loss.py中加入Volga监控预警; 品种选择优先低Volga/Vega比率
